# Get_next_line

Get_next_line is a project for 42 school.

The goal of this project is to create a function that reads a file and returns on each call a new line.

Arg : #1. file descriptor
      #2. The value of what has been read
    
Return value :  1 : A line has been read
                0 : Reading is over
                -1 : An error has occurred
